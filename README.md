# TP Integrador - Organización Empresarial

## Descripción

Proyecto de automatización del proceso de soporte técnico mediante chatbot.

El objetivo es modelar y simular un chatbot capaz de:

- Registrar incidencias.
- Solicitar información al usuario.
- Validar datos.
- Consultar una base de datos simulada.
- Generar tickets de soporte.

## Tecnologías utilizadas

- Python
- Excel (.xlsx)
- BPMN 

## Archivos incluidos

- chatbot_soporte.py
- Base_Datos_Simulada_Chatbot.xlsx
- TP_Integrador.pdf

## Ejecución

```bash
python chatbot_soporte.py
```
